1 . 检查 Node的当前版本，使用命令

````
node -v
````

----

2 . 检查 npm的当前版本，使用命令

````
npm -v
````
----

3 .npm的更新

````
sudo npm install -g latest 
````
----

4 .清除npm cache

````
sudo npm cache clean -f
````
----

5 .安装n模块

````
sudo npm install -g n
````
----

6 .升级到最新版本

````
升级到制定版本：
sudo n 6.5.11
升级到最新的稳定版：
sudo n stable
````
----

7.成功之后，可以用第一条和第二条来检测一下版本即可。
